# Quickstart

Use these commands to work with MindOS on any day:

- `pnpm mcp:dev` — start the MCP runtime for local development.
- `pnpm context:validate` — run strict validation across all context collections.
- `pnpm docs:build` — build or lint the docs tree (stub today, ready for future automation).

Run `pnpm context:validate` before major edits or commits, and keep `pnpm docs:build` handy once a docs pipeline is in place.
