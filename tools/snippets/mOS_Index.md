Usage
- Text Blaze: use the shortcuts below as-is.
- VS Code: same shortcuts work via `.vscode/mindos-all.code-snippets` (type prefix, press Tab). The VS Code-only variant of Research RAW is `/mOS.research.raw.vs`.

---

Workflows – RAW / MOD / OP

Research
- /mOS.research.raw  – dump references, curiosity, unfiltered notes
- /mOS.research.mod  – model actors, motives, constraints, environment
- /mOS.research.op   – run Industry Briefing / research operator

Analysis
- /mOS.analysis.raw  – dump what I’m analyzing + reactions
- /mOS.analysis.mod  – model context using MeyendArcs-style questions
- /mOS.analysis.op   – do a structured breakdown for decisions

Creativity
- /mOS.creative.raw  – capture fragments, vibes, constraints
- /mOS.creative.mod  – define creative objective, intent, constraints
- /mOS.creative.op   – operator for Artistic / Ideation / Hybrid modes

Planning
- /mOS.plan.raw      – brain-dump tasks, obligations, timeframes
- /mOS.plan.mod      – define scope, roles, risks, success criteria
- /mOS.plan.op       – chunk into phases, actions, next moves

Reflection
- /mOS.reflect.raw   – unfiltered emotional / sensory / story capture
- /mOS.reflect.mod   – organize triggers, patterns, axioms, insights
- /mOS.reflect.op    – operator for decode + soothe + integrate

---

Personae – Activators

- /mOS.persona.resonant   – relational, reflective, tone-aware
- /mOS.persona.architect  – systems, structure, planning
- /mOS.persona.alchemist  – experimentation, creative synthesis
- /mOS.persona.executor   – action, constraints, shipping
- /mOS.persona.horizon    – big-picture, future framing, scanning

---

Switches – Global Toggles

- /mOS.switch.quiet   – minimal commentary, just steps/structure
- /mOS.switch.verbose – allow persona voice + short narrative
- /mOS.switch.format  – enforce output format (Markdown, JSON, etc.)

---

Identity – Axioms & Values

- /mOS.identity.axioms – core operating axioms (belief layer)
- /mOS.identity.values – core values + value-friction reflection

---

Regulation – DBT-Aligned Tools

- /mOS.reg.PLEASE      – body/hardware check (sleep, food, illness, etc.)
- /mOS.reg.ACCEPTS     – distress tolerance + distraction toolkit
- /mOS.reg.checkthefacts     – separate facts vs story
- /mOS.reg.oppositeaction    – choose and run Opposite Action
- /mOS.reg.radicalacceptance – Radical Acceptance for stuck realities

---

Reminder:
- Stack: Switch → Persona → Workflow (RAW → MOD → OP).
- When overloaded: start with /mOS.reflect.raw or /mOS.reg.PLEASE.
- When building: set persona + switches first, then fire a workflow.
